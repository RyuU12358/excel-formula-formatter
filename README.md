# Excel Formula Formatter  
Make unreadable Excel formulas *slightly* more readable.

A lightweight web tool that visualizes and formats long Excel formulas.  
It converts deeply nested IF statements and complex expressions into a structured, human-readable tree.  
Supports English and Japanese.

- Format & visualize long Excel formulas
